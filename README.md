# 📚 Verse Villa

### A Community Platform for Readers

**Verse Villa** is a MERN stack application designed for book lovers to connect, create communities, and share their literary thoughts. Join groups, start discussions, and find your next favorite read.

🚀 **Live Demo:** [https://verse-villa.onrender.com/](https://verse-villa.onrender.com/)


---

## ✨ Key Features

* **User Authentication:** Secure Sign Up and Login.
* **Groups:** Create, join, and leave book clubs/interest groups.
* **Posts:** Share thoughts with rich media (image support) inside groups or on the public feed.
* **Responsive Design:** Optimized for desktop and mobile devices.
